# ADR 0003 — Evidence-linked insights

- **Status:** Accepted
- **Decision date:** 2026-07-16

## Context

Music museums benefit from narrative interpretation, but a persuasive interface can make unsupported claims appear factual.

## Decision

Insights are classified as observed, derived, inferred or unavailable. Confidence is calculated from measurable source coverage and evidence, not assigned as decorative copy.

## Consequences

- Every inference-heavy room exposes provenance and limitations.
- Missing granularity remains missing.
- Synthetic histories are only allowed in clearly labelled test fixtures, never user-facing analytics.
- Exact-value tables and exports remain available behind visual summaries.
